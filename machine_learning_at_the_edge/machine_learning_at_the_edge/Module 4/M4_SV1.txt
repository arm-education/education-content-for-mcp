
In this lab, you will learn an efficient


method for processing audio recordings


using mail frequency cepstral coefficients


and a Convolutional Neural Network.


The idea is to segment one slice


of a continuous input signal into


overlapping or non-overlapping segments,


do some preprocessing on each


input slice and stack the resulting


process segments over each other


to obtain a two-dimensional frame.


This allows us to use some of


the same tools that are used for


Machine Learning on real images,


such as using Convolutional Neural Networks.


For this exercise,


we are going to use the "Hey Snips"


dataset from Couke and colleagues.


The train dataset consists of


utterances from 30 speakers,


while the test dataset has


utterances from 10 speakers.


The length of each utterance ranges


between about 3 and 9 seconds.


The data is labeled 1


if the phrase "hey snips" was


uttered and 0 if not.

Next, we extract the features using


Mel-frequency cepstral coefficients.


This state of the art technique


for the speech recognition is


structured around these MFCC features,


which is a nonlinear mapping


of the frequency spectrum to


considerably fewer features.


It is an effective feature


extractor for audio classification.


It converts conventional frequency


measurements to the Mel scale,


which is a logarithmic transformation


that takes into account human


perception at different frequencies.


As MFCC has proven to be a very


good feature for such audio tasks,


there is an integrated function


for it in TensorFlow,


which we'll be using here.
